For in döngüsü

Dart dilinde for in döngüsü, C# dilindeki for each döngüsüne benzemektedir. Yinelenebilir bir nesne üzerinde geçerli yineleme sayacının bilinmesi gerekmediği durumlarda for in döngüsü kullanılabilir. Her yinelemede değişkenin adına döngünün yürütüldüğü nesneden eleman kalmayıncaya kadar bir eleman atanır.

Syntax

for (degisken_adı in nesne)
{
  //yürütülecek kod bloğu
}

Örnek

var list1 = [ 'Dart', 'programlama dili'];
for(var x inlist1)
{ print (x);}
/*
Dart
programlama dili
*/
