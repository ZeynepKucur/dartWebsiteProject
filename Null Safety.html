Dart Null Safety

Null Safety (null güvenliği) Dart 2.12 versiyonu ile birlikte gelmiştir. Dart programlamada dilinde null safety ile bir veritipi null olabileceği belirtilmediği takdirde null olamaz. Bir değişkeni null olabilir (nullable) yapabilmek için değişkenin tanımlandığı veritipinin sağına soru işareti (?) koymak gerekmektedir. Nullable değişkenlere değer atanmadığı sürece Dart default değer olarak null vermektedir.

Örnek

int? sayi; print(sayi); //Çıktı: " null "

sayi değişkeni int? türünde tanımlandığı için null ya da int olabilir ve bu değişkene hiçbir değer atanmadığından sayi değişkeni yazdırıldığında çıktı olarak ‘null‘ verir. Nullable değişkenlere değer verilmediği sürece Dart default değer olarak null vermektedir.

Dart dilinde null nedir?

Null, değeri yok, değersiz, boşluk, hiçlik anlamına gelmektedir. Dart programlama dilinde değişkenlere verilebilen her değer null da dahil bir nesne (object)dir. null dışındaki bütün nesneler nesne sınıfından kalıtım almaktadır. null anahtar kelimesi Null sınıfının tek instance’ı (örneği) olan nesneyi belirtmek için kullanılmaktadır.
