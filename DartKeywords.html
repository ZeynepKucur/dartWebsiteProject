Dart Keywords

Identifier olarak kullanılamayan rezerve edilmiş kelimeler vardır bu kelimeler anahtar kelime (keyword) olarak da bilinir. Anahtar kelimelerin işlevleri derleyici tarafından tanımlanmıştır.

Dart dilinin özel olarak ele aldığı sözcükler:

abstract 2 else import 2 show 1
as 2 enum in static 2
assert export 2 interface 2 super
await 3 extension 2 late 2 sync 1
break external 2 library 2 this
case factory 2 mixin 2 throw
catch false new true
class final null try
const finally on 1 typedef 2
continue for operator 2 var
covariant 2 Function 2 part 2 void
default get 2 required 2 while
deferred 2 hide 1 rethrow with
do if return yield 3
dynamic 2 implements 2 set 2

Bu kelimeleri Identifier (kullanıcı tarafından atanan tanımlayıcı isim) olarak kullanmaktan kaçınılmalı fakat yine de gerekliyse sayılarla işaretlenmiş anahar sözcükler (keywords) Identifier olabilir:

  • Üzerinde 1 yazan sözcükler contextual keyword (bağlamsal anahar kelime)dür. Yalnızca belirli yerlerde anlamları vardır.
  • Üzerinde 2 yazan sözcükler built-in identifier(yerleşik tanımlayıcı)dır. JavaScript kodunu Dart’a taşıma görevini basitleştirmek için bu anahtar sözcükler çoğu yerde geçerli tanımlayıcılardır, ancak sınıf veya tür adları veya içe aktarma önekleri olarak kullanılamazlar.
  • Üzerinde 3 yazan sözcükler eşzamansızlık desteği (asynchrony support) ile ilgili sınırlı ayrılmış sözcüklerdir. async, async*, sync* ile işaretlenmiş herhangi bir fonskiyon gövdesinde await ve yield anahtar sözcükleri identifier olarak kullanılamaz.

Tablodaki diğer sözcüklerin hepsi rezerve edilmiş kelimelerdir, Identifier olarak kullanılamazlar.
