Dart While, do while

while döngüsü

Kod bloğu while döngüsü koşulu doğru olduğu sürece yürütülür. Her yinelemede koşul ifadesi kod bloğu yürütülmeden önce test edilir.

Syntax

while (koşul)
{
  //yürütülecek kod bloğu
}

int x = 3;
while(x<= 5)
{
 print('x = $x');
 x++;
}
/* x = 3
   x = 4
   x = 5 */

do while döngüsü

do while döngüsünde kod bloğu koşul test edilmeden önce bir kez yürütülür yani koşul kod bloğu yürütüldükten sonra test edilir. Koşul kontrol edildiğinde true değer üretirse döngüye devam eder, false değer üretirse döngüden çıkar.

Örnek

int i = 3;
do
{
 print('i = $i');
 i++;
}
  //'i = 0'

Yukarıdaki örnekte görüldüğü üzere, do while döngüsünde ilk olarak kod bloğu yürütülür daha sonra koşul test edilir. Koşulu sağlıyorsa döngü devam eder sağlamıyorsa döngü sona erer.
