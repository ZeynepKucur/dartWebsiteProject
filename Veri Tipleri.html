Dart Veri Tipleri

Veri tipi nedir?

Veri tipleri her dilin en önemli temel özelliklerindendir. Değişken, bir değeri tutmak için kullanılan bir bellek konumuna verilen addır. Veri tipi, değişken tarafından ne tür bir değerin tutulacağını belirtir. Her değişkenin bir veri tipi vardır.

Dart veri tipleri nelerdir?

Dart aşağıdaki yerleşik veri tipleri için özel desteğe sahip:

  • Numbers (int, double)
  • Strings (string)
  • Booleans (bool)
  • Lists (list)
  • Sets (set)
  • Maps (map)
  • Runes (runes)
  • Symbols (symbol)
  • The value null (null)

Aşağıdaki türler de Dart dilinde özel rollere sahip:

  • Object: Null hariç tüm Dart sınıflarının üst sınıfı
  • Future ve Stream: Eşzamansız (asynchrony) destekte kullanılır.
  • Iterable: for-in döngülerinde ve senkron üreteç fonskiyonlarında kullanılır.
  • Never: Bir ifadenin değerlendirmeye hiçbir zaman başarıyla tamamlayamacağını belirtir. Genelde istisna oluşturan fonskiyonlarda kullanılır.
  • dynamic: Statik denetim devre dışı bırakılmak istendiğinde kullanılır.
  • void: Genellikle dönüş türü (return type) olarak kullanılır, bir değerin asla kullanılmadığını belirtir.

Dart Numbers (Sayılar):

Dart dilinde sayılar iki şekilde destekleniyor: int ve double Hem int hem double, num‘ın alt türleridir. num olarak tanımlanan bir değişken hem int hem double değeri alabilir.

int
int veri türü tam sayılar için kullanılır ve değeri platforma bağlı olarak 64 bitten fazla olamaz. native platformlarda değeri -263 ile 263 – 1 arasında olabilir. Web’de ise, tam sayı değerleri JavaScript sayıları olarak gösterilmektedir ve değerleri -253 ile 253 – 1 arasında olabilir.

double
double, IEEE 754 standardı tarafından belirtildiği gibi 64 bit (çift duyarlıklı) kayan noktalı sayılardır.

Strings:

Dart’ta string, bir dizi UTF-16 kod birimini tutar. Dart dilinde char veri tipi bulunmamaktadır. Tek harf bile olsa string veri tipi ile saklanır. Tek tırnak ( ‘ ‘ ) ya da çift tırnak ile ( ” ” ) ile string oluşturalabilir.

Örnek

String s1 = 'Dart öğreniyorum';

String s2 = "Çift tırnakla da string oluşturulabiliyor";

Örnek

String s3 = "Dart'ın özelliği";

String s4 = 'Dart\'ın özelliği';

Booleans

Dart dilinde bool anahtar kelimesi Boolean değerlerini temsil etmek için kullanılır. Sadece true ve false bool türündedir.

Lists:

Nesne koleksiyonunu temsil etmek için list veri tipi kullanılır.Diğer dillerdeki array veri tipine benzer. Dart dilinde array bulunmasa da list array’in sağlayabildiği hemen hemen her şeyi sağlar. Lists sıralı koleksiyon nesneleridir yani her elemanın index ile erişilebilen belirli bir pozisyonu vardır.

Dart dilinde list tanımlama örnekleri:

var list1 = [1 ,2 ,3]

var ile tanımlanmış yukarıdaki kodda Dart type inference ( tür çıkarımı ) özelliği ile list1’in List < int > (int tipinde elemanlı List) olduğu çıkarımını yapar.

var list2 = ['Dart' ,'programlama dili']


Constructor ile List tanımlama:

List constructor ile List tanımlanabilir.

Örnek

var createList = List();

Yukarıdaki şekilde List oluşturulduğunda boş bir List elde edilir.

List içereceği veritipi ile birlikte aşağıdaki şekilde tanımlanabilir:

Syntax

var listAdı = List<veriTipi>();

List veritipi de array veritipi gibi sıfır tabanlı indeksleme sistemini kullanılır. sıfır ilk değerin indeksidir, list.length-1 son değerin indeksidir.

var list3 = [1,2,3,4,5]

print(list3.lengh);//.length listenin uzunluğunu hesaplar.Sonuç 5 olarak çıkar.

print(list3[3]);//3.indeksin değeri olan 4'ü yazdırır.

Dart 2.3 ile versiyonu spread operator (…) ve the null-aware spread operator (…?)‘lerini tanıttı.

spread operator(…) ile bir listenin bütün değerlerini başka bir listeye yerleştirebilirsiniz.

Örnek

list4 = [10, 20];

list5 = [...list4,30, 40, 50];//list4, list5'e dahil edilir.

the null-aware spread operator (…?)operatörün sağındaki ifadenin boş olup olmadığını denetler eğer boşsa o ifadeyi katmaz.

Örnek

varlist6;

varlist7; = [100...?list6];

list6’ya değer atanmadığı için null’dur. …? operatörü kullanılarak null ifade eklemeye çalışmaktan kaynaklanan hatadan kaçınılabilir.

Sets:

Dart dilinde Set benzersiz öğelerin sırasız koleksiyonudur. Tekrarlayan öğeler bulunmaz.

Set-leri tanımlamanın iki yolu vardır.

Örnek

Set <değişken_türü> değişken_adı = {};

var değişken_adı = <değişken_türü>{};

Maps:

Map(harita), anahtarları ve değerleri ilişkirendiren bir nesnedir. Hem anahtarlar hem de değerler herhangi bir nesne türü olabilir. Anahtarların benzersiz olması gerekir.

Örnek

var sayılar={

//anahtar: değer

1: 'bir'

2: 'iki'

3: 'üç' };

Runes:

Stringler bir dizi karakterden oluşur. Dart stringleri Unicode UTF-16 kod birimleri dizisi olarak temsil eder. Unicode dünyadaki tüm yazım sistemlerinde kullanılan her harf, rakam ve sembol için benzersiz bir sayısal değer tanımlayan bir biçimdir. Rune bir Unicode kod noktasını temsil eden tamsayıdır.

Symbols:

Semboller, Dart programında tanımlanmış olan bir operatörü ya da identifier’ı (tanımlayıcı) temsil ederler.Değiştirilemeyen ve derleme süresi boyunca sabit kalan dinamik string adlardır.
