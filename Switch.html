Dart Switch

switch case iç içe if else ifadelerinin basitleştirilmiş halidir. switch case yürütülecek kod bloğunu bulmak için case ifadelerini değerlendirir. Dart dilinde switch döngüsünde ifade ve case’de tamsayılar, stringler ve compile-time constantlar (derleme zamanı sabitleri) karşılaştırılabilir. Karşılaştıralan nesnelerin tümü aynı sınıftan olmalıdır. case sabitleri benzersiz olmalıdır.

Syntax

switch(ifade)
{
  case sabit1:
    //kod bloğu
    break;
  case sabit2:
    //kod bloğu
    break;
  default:
    //kod bloğu
    break;
}

switch case nasıl çalışır?

  • Öncelikle switch parantezi içerisindeki ifade sadece bir kez hesaplanır.
  • switch parentezi içerisindeki ifade her bir case ile kıyaslanır.
  • case ve ifade eşleşirse, o case ifadesiyle ilişkili kod bloğu yürütülür.
  • Hiçbir case ifade ile eşleşmezse default anahtar sözcüğüyle ilişkili kod bloğu yürütülür.

Örnek

int gun = 5;
switch (gun)
{
  case 1:
    Print('Pazartesi');
    break;
  case 2:
    Print('Salı');
    break;
  case 3:
    Print('Çarşamba');
    break;
  case 4:
    Print('Perşembe');
    break;
  case 5:
    Print('Cuma');
    break;
  case 6:
    Print('Cumartesi');
    break;
  case 7:
    Print('Pazar');
    break;
} //Çıktı : Cuma

break anahtar sözcüğü

Akış break anahtar sözcüğüne geldiğinde, switch case döngüsünden çıkılır.
Yukarıdaki örnekte olduğu gibi ifade bir case ile eşleştiğinde, case ile ilişkili kod yürütülür ve break ile switch case döngüsünden çıkılır.

default anahtar sözcüğü

Switch parentezi içerisindeki ifade herhangi bir case ile eşleşmiyorsa akış default anahtar sözcüğüne gelir ve default ile ilişkili kod bloğu yürütülür.
