Dart For döngüsü

Bir kod bloğu belirli sayıda döndürülmek istendiğinde for döngüsü kullanılır. For döngüsü yineleyici olarak bir değişken alır ve ona bir başlangıç değeri atanır ve koşul doğru olduğu sürece döngü gövdesi yinelenir.

Syntax

for (ifade1 ; ifade2 ; ifade3)
{
  //Döngü boyunca yürütülecek kod bloğu
}

İfade1 kod bloğu yürütülmeden önce sadece bir kere çalıştırılır.Bundan sonraki iterasyonlarda (yinelemelerde) ifade1 bir daha çalıştırılmaz.
İfade2 kod bloğunun yürütülmesi için gerekli olan koşulu tanımlar yani İfade2 çalıştırıldıktan sonra true değer üretiyorsa döngü devam eder. Eğer koşul sağlanamazsa döngüden çıkılır.
İfade3kod bloğu yürütüldükten sonra çalıştırılır.

Örnek

for(int i = 0; i<5; i++)
{
 print('Dart eğlenceli');
}
// 5 kere 'Dart eğlenceli' yazar.

int toplam = 0;
  for (int i = 0; i<4; i++) {
    toplam+= i;
    print('Toplam = $toplam');
  }
/* Her yinelemede 'toplam'ın değerini yazar.
Toplam = 0
Toplam = 1
Toplam = 3
Toplam = 6 */
