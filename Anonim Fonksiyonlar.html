Dart Anonim Fonksiyonlar

Dart dilinde her fonksiyonun bir ismi olmayabilir. Aynı zamanda lambda ve closure olarak da bilinen Anonim fonksiyonlar tıpkı diğer fonksiyonlar gibidir ama isimleri ve geridönüş tipleri yoktur. Bir anonim fonksiyon hiç ya da birden çok parametre alabilir. Dart dilinde anonim fonksiyonlar değişkenlere değer ve fonksiyonlara parametre olarak atanabilirler.

Anonim Fonksiyon sözdizimi

(parametreler){
 ...
}

Anonim Fonksiyon örneği

(int yas){
  return 'yaş: $yas';
}

Fonksiyonun geri dönüş tipi geri dönüş değerinden anlaşılır.

Fonksiyonları değişkenlere atama

Değişkene değer atanacağı zaman fonksiyonlar da veri tipi olarak kullanılabilirler.

Örnek

int sayi = 7;
String karsilama = 'Hoşgeldin';
bool isGood = true;
Function carpim = (int x , int y ) {
  return x*y;
};

sayi’nın türü int, karsilama’nin türü String, isGood’un türü bool olduğu gibi carpim değişkeninin türü de Function’dır. İsimli fonksiyonlar değişkenlere atanamazlar ama anonim fonksiyonlar değişkenlere atanabilirler.

İsimli fonksiyonlar değişkenlere atanamazlar

Function fonksiyon = int isimliCarpim(int x, int y){
return x * y;
}; //Hata verir: "Function expressions can't be named."

Fonksiyonlardan fonksiyon döndürmek

Anonim fonksiyonlar, isimli fonksiyonların return output’u olabilirler:

Örnek

Function isimliFonksiyon() {
  return() {
    print('Hey');
  };
}

Yukarıdaki fonksiyonun dönüş değeri Function tipinde bir anonim fonksiyondur.

Anonim Fonksiyonların kullanımı

Örnek

Function fonksiyon = (int x, int y){
  return x * y;
}

Yukarıda Function tipinde carpim değişkeni tanımlı ve bu carpim değişkinene değer olarak bir anonim fonksiyon atalı. carpim değişkenine atalı bu fonksiyonu çağırmak için değişkenin adını yazıp parantez içine argüman yazmak gerekiyor.

print (carpim(3, 5)); //Çıktı: 15
