Dart Assert

Dart assert nedir?

Dart programlama dilinde Assert boolean koşulları test etmek için kullanılan, yararlı bir debugging (hata ayıklama) aracıdır. Assert ifadesi, verilen boolean koşulu yanlışsa yürütmeyi o noktada Assertion Error ile durdurur, boolean koşulu doğru ise yürütme normal bir şekilde devam eder.

Syntax

assert(koşul);

Örnek

var list1 = [1, 2, 3, 4, 5];
assert (list1 is int);

Yukarıdaki kodda assert ifadesinin koşulu olarak list1’in int türünde olup olmadığı test edilir. Sonuç false olduğu için Assertion Error ortaya çıkar.

assert ifadesine mesaj ekleme

assert ifadesine mesaj eklemek için koşula virgül konularak ikinci argüman olarak string eklenir.

Örnek

var list1 = [1, 2, 3, 4, 5];
assert(list1 is int, 'Yanlış');

Eğer verilen koşul doğruysa yürütme devam eder, eğer yanlışsa yürütme verilen mesaj ile birlikte Assertion Error verilerek durdurulur.
