Dart Statically-typed&Dynamic

Dart, statik tipli bir programlama dilidir.

Statically-typed

Statik tipli programlama dillerinde değişkenin türü compile-time’da(derleme zamanında) bilinir. Veri türü atanmış değişkenin, sonradan veri türünün değiştirilmesine izin verilmez, compile-time error ortaya çıkar.

Örnek

int x = 10; // x adındaki değişkene int türünde değer verildi.

x = 'Dart';/* compile-time error. int türündeki bir değişkene string türündeki bir değer atanamaz.*/
