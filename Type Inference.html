Dart Type Inference

Type Inference (Tür çıkarımı) nedir?

Type Inference (Tür çıkarımı), bir programlama dilindeki bir ifadenin veri türünün otomatik olarak çıkarılmasını ifade eder.

Syntax

Type Inference, veri türü açıkça belirtilmeden bir değişkenin tanımlanabilmesine olanak sağlar. Bunun için veri türü yerine var anahtar kelimesi kullanılabilir. var anahtar kelimesi kullanıldığı zaman Dart o değişkene karşılık gelen verinin tür çıkarımını yapabilir ve değişkene o veri türünü atar. Burada da type safety (tür güvenliği) geçerlidir.

Örnek

var x = 100;

Dart x’in int türünde olduğu çıkarımını yapar. Dart’ın yaptığı tür çıkarımını aşağıdaki şekilde görebilirsiniz:

Örnek

var ad = 'Dart';

print(ad.runtimeType);

// Çıktı: String

Örnek

var sayi = 'on'; sayi = 10;

Dart dilinde type safety (tür güvenliği) olduğu için yukarıdaki kod hata verir. sayi değişkeni var ile tanımlansa da tür çıkarımı ile sayi değişkeninin string tipinde olduğu anlaşılır. Bu değişkene daha sonra int tipinde bir değer atanmaya çalışıldığında Dart type safety buna olanak vermez.
