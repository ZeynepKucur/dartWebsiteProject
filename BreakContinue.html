Break, Continue

break keyword

Döngü içerisindeki break anahtar sözcüğü, o döngünün yürütülmesini durdurmaya ve akışı döngüden çıkartmaya yarar. Program döngü bloğundan sonraki ilk deyimle akışına devam eder.

Örnek

for (int = 0; i < 5; i++)
{
    if (i == 3)
   {
     break;
   }
  print ('Döngü içi, i = '$i ');
}
print('Döngüden çıkıldı');

/* Döngü içi, i = 0
Döngü içi, i = 1
Döngü içi, i = 2
Döngüden çıkıldı.

continue keyword

continue anahtar kelimesi, belirtilen koşul gerçekleştiğinde döngünün o anki iterasyonu sonlanır ve döngünün diğer iterasyonlarıyla devam edilir.

Örnek

for (int i = 0; i <=5; i++)
{
  if (i == 4)
  {
    continue;
  }
  print('i = '$i ');
}
/* i = 0
i = 1
i = 2
i = 3
i = 5 */
