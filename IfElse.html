Dart If Else

if deyimi

If deyimi, yalnızca belirtilen koşul doğru olduğunda kod bloğunun çalıştırılmasına izin verir.

Dart dilinde en genel if deyimi aşağıdaki gibi yazılır:

if (koşul) {

  //koşul doğru ise çalıştırılacak kod

}

Örnek

if ( 10 > 5 )
{
  print('on, beşten büyüktür.');
}

If deyimi nasıl çalışır?

Programın akışı if deyiminin olduğu satıra geldiğinde if parantezi içindeki, true ya da false değer üretecek koşul ifadesi çalıştırılır. Koşul true değer üretirse yukarıdaki örnekte olduğu gibi kod bloğu içindeki komut çalıştırılır değilse çalıştırılmaz.

else deyimi

else deyimi eğer if deyiminin içindeki koşul false değer üretirse başka bir komut çalıştırmak için kullanılır.

Örnek

if (koşul)
{
  //Komut 1: Koşul doğru ise çalıştırılacak komut
}
else
{
//Komut 2: Koşul yanlış ise çalıştırılacak komut
}

Örnek

int yas = 15;

if (yas >= 18 )
  print ('yetiskin');
else
  print ('cocuk');
//'çocuk' çıktısını verir.

Yukarıdaki örnekte olduğu gibi eğer komutlar tek satırdan oluşuyorsa süslü parantez içine almak zorunlu değildir. yas değişkeni 15 olduğu için if parantezi içindeki koşul ifadesi false değer üretir ve else deyimi için olan komut çalışır.

if else if deyimi

if else if deyimi test edilecek farklı koşullar eklenmesine olanak sağlar.

Syntax

if (koşul1)
{
  //Komut 1: Koşul doğru ise çalıştırılacak komut
}
else if (koşul2)
{
  //Komut 2: koşul1 yanlış ve koşul2 doğru ise çalıştırılacak komut
}
else
{
 //Komut 3: koşul1 ve koşul2 yanlış ise çalıştırılacak komut
}

Örnek

int saat = 15;

if(saat == 10)
{
  print('Günaydın');
}
else if (12<saat<18)
{
  print ('iyi günler');
}
else
{
  print('iyi akşamlar');
}
// 'İyi günler' çıktısını verir.
