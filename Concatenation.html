Dart Concatenation (String birleştirme)

Concatenation nedir?

Concatenation Türkçe’ye birleştirme olarak çevrilebilir. Peki concatenation nedir? Dart programlama dilinde stringleri birleştirme eylemine concatenation denir. Stringleri birleştirmenin birkaç yolu vardır.

+ operatörü

+ operatörü ile stringler birleştirilebilir.

Örnek

print( 'Dart' + 've' + 'Flutter' + 'öğrenelim' );
//Çıktı : "DartveFlutteröğrenelim"

+ operatörü kullanmadan bitişik string literallar ile de aynı işlem görülebilir.

Örnek

print('Dart' 've' 'Flutter' 'öğrenelim');
//Çıktı : "DartveFlutteröğrenelim"

Interpolation

Dart programlama dilinde Interpolation ile bir değişkenin değeri bir expression ile birleştirilebilir.
Expression’ın değeri string içerisine ${expression} şeklinde konulabilir.
Expression bir Identifier ise süslü parentezlere gerek yoktur, $identifier şeklinde konulabilir.

Örnek
String s1 = 'interpolation';

print('Dart dilinde $s1 vardır.');
//Çıktı: "Dart dilinde interpolation vardır."
Örnek
String s1 = 'interpolation';

print(print('Dart dilinde {$s1.toUpperCase()} vardır.');
//Çıktı: "Dart dilinde INTERPOLATION vardır."

Tırnak İşareti

Dart programlama dilinde çok satırlı string, üçlü tek tırnak ( ‘ ‘ ‘ ) ya da üçlü çift tırnak ( ” ” ” ) ile oluşturulabilir.

Örnek

String cokluSatir = '''
Bunu yaparak
çok
satırlı
stringler
elde edebilirsiniz.''' ;
print(cokluSatir);
/*'''
Bunu yaparak
çok
satırlı
stringler
elde edebilirsiniz.'''*/

Raw String

Dart programlama dilinde raw stringlerde \ gibi karakterler özel muamele görmezler, string içerisine yazılan her karakter yazıldığı gibi çıkar. Raw String elde edebilmek için tırnak işaretinin başına r harfi konulmalıdır.

Örnek

String rawString = r'Raw stringlerde \n alt satıra geçirmez.';

print rawString();
//Çıktı: "Raw stringlerde \n alt satıra geçirmez."
